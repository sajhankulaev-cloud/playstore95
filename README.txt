PlayStore 95 — PS Store importer (stable)

Функции:
- Каталог (36 на странице, поиск, сортировка по цене)
- Админка: импорт игры по ссылке PS Store, курсы, настройки WhatsApp, список игр и удаление

Запуск:
1) .env.example -> .env
2) npm install
3) npx playwright install chromium
4) npm run dev

Админка: http://localhost:3000/admin
